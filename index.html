My Journey

Coding was not in my mind when I was a working journalist a few years ago. I did not consider it necessary to know how to code or how to build an interactive website. At that time, web developers worked in the media industry to maintain the websites. But they were not journalists.

But things have changed lately. Coding and programming are now considered relevant to the media industry while many media schools have started offering coding and programming courses.

I became interested in coding a few years ago. However, I neither had time nor I could find an appropriate course to learn coding. Since I'm stuck at home this summer due to COVID situation, this was the perfect time for me to start learning coding.

The first two weeks of this Web Development course was challenging for me. I was overwhelmed with the amount of information on Blackboard. It probably happens for most online classes. But as I started building my website, I became more comfortable with the web development process.
